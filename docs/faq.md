Common background notes for readers who are new to the project.

## What is Kinoshita Electronics Consortium and the [KN-86 Deckline](https://kn86-deckline.com/)?

Kinoshita Electronics Consortium is a fictional cyber terminal manufacturer from
Kinoshita Sekai, a world imagined by Josh Schairbaum and Great Western
Productions. The KN-86 Deckline is one of its handheld terminals and uses KEC
Lisp as its scripting language; this repository contains the standalone language,
standard library, command-line tool, and tests that can run on a normal computer.

## Is this a real language?

Absolutely. KEC Lisp is intended for use in day-to-day projects, most notably
the cartridge authoring system of the KN-86 Deckline.
